# YouTube Playlist Downloader
Download a YouTube playlist through CLI

## Install
Install the CLI globally
```
npm i -g ytdx
```

## Usage
You will need a directory (folder) and a public YouTube playlist
```
ytdx --dir DIRECTORY_NAME --id PLAYLIST_ID
```
